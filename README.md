# Customer Churn Analysis Project

## Objective
To analyze customer churn and identify factors causing customer attrition.

## Tools Used
- Excel
- GitHub

## Dataset
Customer demographic, tenure, contract type, and churn status.

## Analysis Performed
- Churn rate calculation
- Churn by contract type
- Customer tenure analysis

## Key Insights
- Month-to-month customers show higher churn
- Customers with low tenure are at higher risk
- Long-term contracts improve retention

## Conclusion
This project demonstrates basic data analysis and business insight generation.
